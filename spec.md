# Specification

## Summary
**Goal:** Fully redesign the visual theme of the Khudrang Kalakaar Portfolio website with a bold, artisanal mural-artist aesthetic using a warm earthy color palette (terracotta, cream, charcoal, deep earth tones) applied consistently across all sections.

**Planned changes:**
- Update global CSS and Tailwind config with new color tokens, typography, and spacing utilities reflecting the artisanal theme
- Redesign Navigation with refined logo presentation, cleaner link styling, polished mobile menu, and prominent "Hire Me" CTA button
- Redesign HeroSection with full-screen immersive layout, dramatic large typography, and well-composed background image overlay
- Redesign AboutSection with improved two-column layout, portrait prominence, floating experience badge, and stats row
- Redesign ServicesSection with polished card layouts, refined borders, shadows, icons, and hover effects for all seven services
- Redesign FeaturedProjects and ProjectSection with editorial-style image/text split, styled numbered badges, and improved section heading
- Redesign ArtworkGallery with a clean responsive grid, elegant card hover effects, and polished lightbox dialog
- Redesign WhyChooseSection with refined trust-point cards, styled numbered accents, and updated section heading
- Redesign ContactSection with polished Location, WhatsApp, and Instagram contact cards with refined iconography
- Redesign Footer with improved layout, typography, and visual hierarchy for brand info, nav links, and contact details
- Preserve all existing backend data-fetching logic, double-tap logo easter egg, and WhatsApp pre-filled message links throughout

**User-visible outcome:** The entire portfolio website presents a cohesive, bold, artisanal visual identity with warm earthy tones and strong typography — all sections feel unified and professionally redesigned while all existing functionality remains intact.
